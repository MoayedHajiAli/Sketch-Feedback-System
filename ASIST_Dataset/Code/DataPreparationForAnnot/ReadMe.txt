This folder includes tools for preparation of sketch data for annotation. 

convertTabs.m : 	Converts tabs to spaces or vice-versa. Necessary to parse sketches into a single format when their formats varies.
getAllFiles.m : 	Returns all files in a folder.
transferFiles.m : 	Copies all the files inside of a (sub)folder to a directory.
xmlToPNG.m : 		Converts sketch data into images and saves to the file.
